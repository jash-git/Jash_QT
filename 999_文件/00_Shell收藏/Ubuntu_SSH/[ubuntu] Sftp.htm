[ubuntu] Sftp – 透過Pietty、Filezilla來使用 Sftp

2010.07.31 / 分類: Linux / 1 則回應

標籤: Ubuntu, 主機, 研究生生活

有時候只是想傳個小小的設定檔到 Server 去

除了手動key以外

應該不會有人像我一樣把檔案透過有裝 FTP 套件的 Server

再將檔案移到 www 目錄去

再使用 wget 去做下載吧?

之前就聽過可以使用sftp了

只是沒有時間去試

沒想到還蠻簡單的!

只要 Server 有安裝 SSH 就可以使用 Sftp 了

一、Pietty

首先先打開 pietty 軟體

輸入 主機位置 跟 port

之後按連線

進去以後先看一下資料夾內容~

很好~

是空的!

之後直接把要上傳的檔案

直接拉到視窗裡面

之後就會跳出這個訊息

這邊要注意一下

在 /tmp 前面有個 . (點)

這代表是 egg 這使用者的家目錄

因為我檔案要傳到家目錄 tmp 底下

所以我在後面加上 /tmp

之後輸入使用者的密碼

然後就上傳成功囉!!

再看一下目錄現在的情況

剛剛上傳的檔案就在裡面了!


二、Filezilla

打開站台管理員以後

按下新增站台

將需要的資訊都填一填

重點在於伺服器種類要選擇 SFTP – SSH File Transfer Protocol

之後就可以連線囉!!

輸入使用者的密碼

連線成功以後

接下來就像是在使用一般的FTP一樣

上傳檔案囉!!
