網頁的目標位置(TARGET)
通常使用在框架模式或是另外開視窗的狀態下，主要的使用方法為：
  • TARGET="框架名稱"
    其參數如下
  • target="_blank"
    開一個新的視窗顯示網頁內容。

  • target="_parent"
    將連結的畫面內容，當成文件的上一個畫面。

  • target="_self"
    將連結顯示在目前的視窗之中。

  • target="_top"
    連結顯示時占滿整個視窗。

    範例：<a href="href.htm" target="_blank">開新視窗</a>

    網頁內部的連結製作法
    先在連結處作記號：
    <A NAME="A1">想連結的點</A>
    設定連結：<A HREF="#A1">連結</A>
    進階技巧-連結介紹-JAVASCRIPT
    範例：
    <A href="href.htm" OnmouseOver="window.status='這裡可以填入網頁介紹的內容，滑鼠移過時就會顯示在下面的狀態列喔。';return true;" OnMouseOut="window.status=' ';">把滑鼠靠過來看看</A>

    結果：
    把滑鼠靠過來看看
    進階技巧-連結變色-CSS(Cascading Style Sheets)
    把下面的這堆文字複製起來，貼在</HEAD>之前就可以啦。

    <STYLE type=text/css>
    A:link {COLOR: #0000ff; FONT-SIZE: 12pt; TEXT-DECORATION: none; font-family: 標楷體}
    A:visited {COLOR: #0000ff; FONT-SIZE: 12pt; TEXT-DECORATION: none; font-family: 標楷體}
    A:ACTIVE {COLOR: #00FFff; FONT-SIZE: 12pt; TEXT-DECORATION: none; font-family: 標楷體}
    A:hover {BACKGROUND-COLOR: red; COLOR: white; FONT-SIZE: 13pt; TEXT-DECORATION: none; font-family: 標楷體}
    </STYLE>
