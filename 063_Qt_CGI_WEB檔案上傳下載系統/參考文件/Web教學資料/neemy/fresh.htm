自動換頁喔，經過一段時間就可以換到別的網頁喔，
It's Automatic!
使用方法很簡單，還記得你寫第一篇網頁的時候的開頭嗎？忘記了？我再打一次給你看。有沒有看到 有一串字是<META HTTP-EQUIV....的嗎？我們換頁的指令就是下在這裡喔。

<HTML>
<HEAD>
<META HTTP-EQUIV="Content-Type" CONTENT="text/html; charset=big5">
<META NAME="Author" CONTENT="這裡可以寫入作者的名字">
<TITLE>網頁標題要寫在這裡</TITLE>
</HEAD>
<BODY>
這裡就是你要編輯網頁的地方，也就是主要的精華喔。
</BODY>
</HTML>

我們現在把換頁指令
<META HTTP-EQUIV="Refresh" CONTENT="23; URL=換頁後連結的網址">
加進去囉，所以變成下面的情況。「CONTENT="23;」，23是代表經過23秒後換到別的網頁。這樣了解嗎？

<HTML>
<HEAD>
<META HTTP-EQUIV="Content-Type" CONTENT="text/html; charset=big5">
<META HTTP-EQUIV="Refresh" CONTENT="23; URL=換頁後連結的網址"> <META NAME="Author" CONTENT="這裡可以寫入作者的名字">
<TITLE>網頁標題要寫在這裡</TITLE>
</HEAD>
<BODY>
這裡就是你要編輯網頁的地方，也就是主要的精華喔。
</BODY>
</HTML>
