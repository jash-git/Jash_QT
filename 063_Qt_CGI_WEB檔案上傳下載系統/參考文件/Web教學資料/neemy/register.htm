網站做好了沒人參觀？沒關係，登錄到國內各大網站就會有人參觀了喔。
通常要登錄各大網站有幾項必須的步驟。
  1. 在你要登錄的網站的搜尋器裡面輸入你的網站名稱。如果有重複的名稱，就必須換一個名稱，如果找不到這個資料，你就可以進行第二步驟。
  2. 選擇你的網站主題，例如你的網站專門介紹植物，你就可以在搜尋器之內輸入植物，看看查詢的結果，比對結果發現植物是屬於哪個目錄的，進入那個目錄。
  3. 選擇登錄、介紹網站、或是推薦網站。
  4. 輸入你的網站資料、內容、網址、主題、聯絡信箱、網站主人...
  5. 確定正確後送出，大概一星期左右你的網站就會有人進來參觀了喔。
