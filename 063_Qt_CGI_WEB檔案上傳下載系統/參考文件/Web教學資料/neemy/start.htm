都準備好了ㄏㄛ...我們要開始寫網頁囉！
  • 首先，先設定你要設計的網頁主題。例如我要做的網頁主題是「我的第一個網頁」。
  • 開啟你要寫網頁的軟體(記事本)，從 開始→程式集→附屬應用程式→記事本 就可以開啟囉！
  • 準備好了沒？Let's GO!
1.首先，先把下面的網頁碼複製起來，按滑鼠右鍵複製就可以囉。
<HTML>
<HEAD>
<META HTTP-EQUIV="Content-Type" CONTENT="text/html; charset=big5">
<META NAME="Author" CONTENT="這裡可以寫入作者的名字">
<TITLE>網頁標題要寫在這裡</TITLE>
</HEAD>
<BODY>
這裡就是你要編輯網頁的地方，也就是主要的精華喔。
</BODY>
</HTML>
2.然後把剛剛複製的貼到你的記事本中(按右鍵選擇貼上就可以囉。)


然後接下來的事情很重要喔，要注意看喔。就是把你剛剛編輯的網頁存檔起來。你可以選記事本裡面的檔案→儲存檔案。要記的存檔的時候檔案的副檔名必須為 .htm 或是 .html 喔，要記得打副檔名喔。通常主機的首頁檔案名稱為 index.htm 或是 index.html 。如果你的網頁是首頁(進去第一個看到的網頁)，一定要存成 index.htm 或 index.html 喔。
再存檔之前我們要先建立一個目錄，要不然等一下會找不到自己把網頁都在哪了！我們就先建立mypage這個目錄，我們把網頁通通放在這個目錄喔。
標籤解釋    
這就是最簡單的網頁啦！有沒有發現，其實網頁只是一堆標籤的組成，只不過後面的標籤要多加一個"/"。(<body>網頁主要內容</body>)。通常HTML一個標籤代表一段文件的開始，在標籤內加入"/"代表這份文件的結束。除了少部分的標籤(<br>..)外，其他的都要有結束的標籤，要不然文件很容易出錯喔。

現在就幫你介紹一下這個網頁中標籤代表的意義。
  • <HTML>
    這個就是告訴瀏覽器「我是一個網頁文件」的標籤啦。

  • <HEAD>
    這個是網頁開頭輸入的資訊，告訴瀏覽器網頁主要的資訊。

  • <META HTTP-EQUIV="Content-Type" CONTENT="text/html; charset=big5">
    這個是告訴瀏覽器「我是中文的網頁喔，不要用其他語言來顯示我」，big5就是繁體中文的意思啦。

  • <META NAME="Author" CONTENT="這裡可以寫入作者的名字">
    這個是要讓別人知道網頁作者是誰的啦。不加也沒關係。

  • <TITLE>網頁標題要寫在這裡</TITLE>
    這個就是網頁的標題囉，也是告訴別人網頁內容主要寫蝦米碗糕的地方。

  • </HEAD>
    有開始就要有結束，通常這個標籤會放在網頁的最前面。

  • <BODY>
    這裡就是你要編輯網頁的地方，也就是主要的精華喔。，你可以在這裡加上背景、顏色、影像、連結.....

  • </BODY>
  • </HTML>
    這兩個標籤代表整個網頁文件的結束，通常是放在網頁的最後面。
