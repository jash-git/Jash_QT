寫了網頁放在家裡給自己看？不好吧！傳到網路上給每個人看才對吧。因為每個FTP軟體操作都不太一樣，但是使用方法都一樣，所以這邊用文字敘述的告訴你怎麼上傳喔。
使用CUTEFTP上傳。


  1. 首先先找到CUTEFTP這套軟體，安裝在電腦裡面(可以在TOGET或是ALLFILE找到)

  2. 安裝完了開啟程式，你會發現有個登入的畫面(正常情況下的話)，找到「」這個按鈕，然後按下去。

  3. 按下去之後會出現一個英文的畫面，左邊的畫面如下
    Site Label
    Host Address
    User IDPassword


  4. 輸入完後按確定，然後會發現連線的地方多了你剛剛輸入的名稱，直接在上面點兩下，就可以開始連線囉。

  5. 開始連線時，CUTEFTP上面會有指令的動作，如果有看到SUCCESSFUL的文字代表連線成功囉。連線成功後，下面的視窗左邊是我們的磁碟機，右邊是放網頁的主機。

  6. 從左邊進入剛剛我們放網頁的地方，然後右邊進入主機規定讓你放網頁的地方。

  7. 從左邊的視窗中選取檔案，然後按滑鼠左鍵兩下就可以傳上去囉！如果要一次選很多檔案，先把鍵盤上左下角的Shift按著不放，然後用滑鼠上下移動就可以啦。如果要選特別的檔案，先把鍵盤上左下角的Ctrl按著不放，然後用滑鼠點你想要的檔案就可以啦，然後直接用"拖"的拖曳到右邊就可以囉。

  8. 連上你的個人網頁欣賞成果囉^_^。

  9. 注意，首頁檔案一定要命名為 index.htm 或是 index.html ，否則主機會不知道要顯示哪個檔案唷。

  10. 如果主機已經有相同的檔案，你上傳上去主機會問你SKIP(跳過，主機端的檔案不作任何更改)，OVERWRITE(覆寫，也就是取代主機的檔案)，RENAME(改名字)。

  11. 差不多就是醬子啦！祝你成功囉！
