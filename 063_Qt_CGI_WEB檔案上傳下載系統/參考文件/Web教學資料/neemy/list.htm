麼叫做序列標籤？！其實網頁終場常看到，呈現的形式就像是把文字一行一行的列出來，讓版面看起來更為整齊。其中又分為「無序列標籤」和「有序列標籤」。

無序列標籤...其實就是排列的時候前面的符號是固定的，不會有1、2、3....或是a、b、c...等排列的現象。<UL>即是無序列標籤，每增加一列就必須增加一個<LI>
標籤原始碼顯示結果
<UL>
<LI>我最大
<LI>我是老二
<LI>我排老三
<LI>我最小
</UL>
  • 我最大
  • 我是老二
  • 我排老三
  • 我最小
 
 
有序列標籤.....就是會幫你排列1、2、3...或是a、b、c...的喔。
標籤原始碼顯示結果
<OL>
<LI>我最大
<LI>我是老二
<LI>我排老三
<LI>我最小
</OL>
  1. 我最大
  2. 我是老二
  3. 我排老三
  4. 我最小
我要從特定的數字開始顯示！假設我要從5開始計數
