function Act(menuName)
{
if(document.images){
document[menuName].src = eval(menuName+".src")
}
}
function Inact(menuName)
{
if(document.images){
document[menuName].src = eval(menuName+"off.src")
}
}

修改連結
把連結後面變成下面的形式就可以囉！要記得改連結中的 menu0 改成你宣示的圖檔編號喔。
<A HREF="連結網頁.htm" onMouseOut="Act('memu0')" onMouseOver="Inact('memu0')">
<IMG NAME="memu0" SRC="000a.gif" BORDER="0"></A>



如果不想要有連結，直接把標籤原始碼改成
<IMG NAME="memu0" SRC="000a.gif" BORDER="0" onMouseOut="Act('memu0')" onMouseOver="Inact('memu0')"></A>
就可以啦！趕快試試看喔。
