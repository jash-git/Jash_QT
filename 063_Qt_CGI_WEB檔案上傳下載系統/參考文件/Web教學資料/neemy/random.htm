隨機提示文字
把下面的程式碼複製後貼到網頁上就可以啦。

<SCRIPT Language="JavaScript">
tips = new Array(6);//(6)是代表全部隨機撥放的數目，
tips[0] = "我是第一個！";
tips[1] = "我可以提示文字。";
tips[2] = "我可以貼圖。";
tips[3] = "我還可以撥音樂。";
tips[4] = "我可以做很多事。";
tips[5] = "可是我不能生一個寶寶給你。";
index = Math.floor(Math.random() * tips.length);
document.write("<CENTER>" + tips[index] + "</CENTER><P>");
</SCRIPT>


應用篇--隨機撥放背景音樂。
 <SCRIPT Language="JavaScript">
tips = new Array(6);//(6)是代表全部隨機撥放的數目，
tips[0] = "音樂.mid";//可以使用連結喔。
tips[1] = "音樂1.mid";
tips[2] = "音樂2.mid";
tips[3] = "音樂3.mid";
tips[4] = "音樂4.mid";
tips[5] = "音樂5.mid";
index = Math.floor(Math.random() * tips.length);
document.write("<EMBED autostart=true HIDDEN=TRUE LOOP=TRUE SRC=" + tips[index] + "></EMBED><P>");
</SCRIPT>


應用篇--隨機撥放影像。
<SCRIPT Language="JavaScript">
tips = new Array(6);//(6)是代表全部隨機撥放的數目，
tips[0] = "影像1";//可以使用超連結喔。
tips[1] = "影像2";
tips[2] = "影像3";
tips[3] = "影像4";
tips[4] = "影像5";
tips[5] = "影像6";
index = Math.floor(Math.random() * tips.length);
document.write("<IMG HEIGHT=72 WIDTH=72 SRC=" + tips[index] + "></img><P>");
</SCRIPT>
