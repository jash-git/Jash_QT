通常網路上常見的圖檔的格式為JPG和GIF檔，其中GIF只有256色，但文字比較清晰，還可以做成動畫，相對的JPG為全彩，容量小，但是因為是採用破壞式壓縮，所以文字和圖像容易模糊掉。在網路的世界裡，檔案小、讀取快、網頁美觀是一個吸引別人的重要因素喔。美工軟體我都是用PHOTOIMPACT 哦。

語法標籤介紹
  • <IMG SRC=model.gif>
    這就是貼圖的最最最最最基本的指令啦！最簡單的。<IMG SRC= 部分就是貼圖指令，model.gif就是你貼圖的檔案名稱囉！注意！在網頁中的檔案名稱大小寫是有差別的喔。

  • <IMG SRC=model.gif border="0">>
    這是把圖案邊邊藍色的框去掉的指令，有時候如果貼圖被拿來作超連結的按鈕，旁邊會多一個醜醜的框，用 border="0" 就可以把框框消掉囉。border 就是框框的意思，0 代表沒有，數字越大框框就越大囉。

  • <IMG SRC="model.gif" height="66" width="72">
    這個是設定圖案大小的指令，可以省去瀏覽器幫你計算圖檔格式的時間(雖然感覺不出來)，也可以自由規劃圖檔的大小格式。其中其中HEIGHT代表圖案的高度，WIDTH代表圖案的寬度。

  • <IMG SRC="model.gif" height="66" width="72" alt="圖檔介紹內容">
    ALT指令是說明圖檔的名稱的，如果有時候圖檔不見了，或是還沒下載完成，可以藉由這個指令事先說明圖檔的名稱或是內容。

  • <IMG SRC="model.gif" align="right">
    ALIGN在貼圖中是代表文字和圖案的相對位置，使用方法為 ALIGN="方位"(看上面就知道啦^_^)

  • <IMG SRC="model.gif" VSPACE="15">
    VSPACE是設定圖案旁邊的文字跟圖案本身的垂直距離，HSPACE是設定圖案旁邊的文字跟圖案本身的水平距離，有時一堆字擠在圖案旁邊真的不是普通醜說。

  • <br><IMG SRC="model.gif" VSPACE="15"><br>
    <br>是強迫換行的指令，每輸入一次<br>，就可以強迫文件換一行喔！很好用，常常應用在文件中喔。
