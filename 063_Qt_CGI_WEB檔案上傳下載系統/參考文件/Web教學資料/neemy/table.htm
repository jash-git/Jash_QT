標籤解釋
  • <TABLE BORDER="1">
    告訴瀏覽器，我現在要開始作一個表單了喔。
  • <TR>
    我現在要開始作第一層囉。
  • <TD>
    我現在要開始作第一層裡面的第一格囉。
  • </TD>
    我第一層裡面的第一格已經做完囉。
  • </TR>
    我第一層已經做好囉。
  • </TABLE>
    這個表單我做完囉。

  • 其實大家覺得很複雜對不對！那我們現在把表單當作蓋一間房子好了，房子裡面的房間每間都一樣大。
    1. 好啦，現在我們要開始蓋房子啦，我們用<TABLE BORDER="1">告訴瀏覽器我們開始蓋房子啦。
    2. 然後房子都是一層一層蓋的啊，那我們用<TR>告訴瀏覽器我們要蓋第一層囉。
    3. 瀏覽器現在已經知道我們要蓋房子囉，而且要蓋第一層，那現在我們要開始蓋第一層的第一個房間囉，我們用<TD>告訴瀏覽器，我們要蓋第一層房間囉。過了很久，好不容易第一間蓋完啦，我們用</TD>告訴瀏覽器我們蓋完第一間囉。
    4. 我們現在突然不想蓋囉！我們要告訴瀏覽器我們不蓋囉，我們用</TR>告訴瀏覽器我們不蓋這層囉，用</TABLE>告訴瀏覽器我們房子已經蓋完囉。
