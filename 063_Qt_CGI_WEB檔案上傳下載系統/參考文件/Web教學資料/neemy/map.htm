影像地圖
網頁上常常看到會看到一張圖片，然後當滑鼠移動到圖片上面的某個地方就會有連結，按一下就會連結到其他的地方，這就是影像地圖啦。版豬教你如何使用影像地圖。一大沱連結呢？該如何在自己的網頁中加上連結呢？

這個就是利用影像地圖一個例子喔。
我們要如何做一個影像地圖呢？
  1. 先準備一張圖案。
  2. 計算好圖案的座標位置(可以用PHOTOIMPACT找)
  3. 開始編輯網頁標籤(如下)

    <map name="地圖名稱">
    <area shape=形狀 coords="座標" href="連到哪裡去">
    <area shape=形狀 coords="座標" href="連到哪裡去">
    ...
    </map>
    <img src="xxx" usemap="#地圖名稱">
    這樣就可以構成一個影像地圖了喔。

    影像地圖中的形狀可以是矩形(rect)、圓形(circle)、以及多邊形(polygon)。
    矩形的座標寫法是：
    〝左上角座標，右下角座標〞
    <area shape=rect coords="25,20,75,82" href="rect.htm">

    圓形的座標的寫法是：
    〝圓心座標(x,y)，半徑〞
    <area shape=circle coords="130,50,25" href="circle.htm">


    多邊形﹝三角形也算是多邊型﹞的座標的寫法是：
    〝第一點座標，第二點座標，...，第N點座標〞
    <area shape=polygon coords="203,28,232,80,180,80" href="triangle.htm">
    <area shape=polygon coords="272,30,305,30,318,43,292,76,260,47" href="poly.htm">


    <map name="demo">
    <area shape=rect coords="10,20,30,35" href="rect.htm">
    <area shape=circle coords="100,50,25" href="circle.htm">
    <area shape=polygon coords="200,20,230,80,180,80" href="poly1.htm">
    <area shape=polygon coords="270,30,300,30,310,40,290,70,260,40" href="poly.htm">
    </map>
    <img src="map.jpg" usemap="#demo" border=0>
    這樣會用了嗎？還不會？多試試幾次就知道囉。
進階利用-為圖形上的連結加上說明
大家還記得在聯結標籤的地方是不式有一個地方告訴我們如何介紹連結？沒錯，這裡也可以用到。不過有一點點不一樣喔。 我們把後面的
return true" OnMouseOut="window.status='' 都去掉囉。

使用語法如下：
<map name="demo">
<area shape=rect coords="10,20,30,35" href="rect.htm" onMouseOver="window.status='說明';>
<area shape=circle coords="100,50,25" href="circle.htm" onMouseOver="window.status='說明';>
<area shape=polygon coords="200,20,230,80,180,80" href="poly1.htm" onMouseOver="window.status='說明';>
<area shape=polygon coords="270,30,300,30,310,40,290,70,260,40" href="poly.htm" onMouseOver="window.status='說明';>
</map>
<img src="map.jpg" usemap="#demo" border=0>

有沒有發現規律呢？沒錯，直接把
onMouseOver="window.status='說明';加在連結後面就可以了！這是不是跟連結標籤裡面的很像呢？
