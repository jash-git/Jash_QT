框架就是 FRAME，用在很多地方喔，像你現在用的網頁模式其實就是一個框架的模式，不信你把滑鼠網左邊移動看看，會有一個地方出現←→喔，可以拉動改變視窗。不過如果框架沒有設定好，那參觀者可能就看到空白的網頁囉。如果應用的好，那就可以免去看一個網頁換一個網址的麻煩喔。

好啦，既然我們已經知道框架是什麼東西，那我們就開始使用框架囉。首先，先建立一個HTM檔案囉，隨便你命名啦！把下面這對原始碼複製並貼到你新開的文件中喔。
<HTML>
<HEAD>
<META HTTP-EQUIV="Content-Type" CONTENT="text/html; charset=big5">
<META NAME="Author" CONTENT="這裡可以寫入作者的名字">
<TITLE>我也會作框架喔</TITLE>
</HEAD>
這裡就是你要編輯網頁的地方，也就是主要的精華喔。
</HTML>

奇怪？不是要加<BODY>嗎？怎麼不見了？不用擔心啦，在框架的文件中，FRAMESET會取代BODY的啦。

在上面我們看到的，<FRAMESET ROWS="100,*">中的
FRAMESET就是告訴瀏覽器我們要分割視窗了，
COLS是告訴瀏覽器我們要分割成左右兩部分，
ROWS是告訴瀏覽器我們要分割成上下兩部分，後面的數字是代表分割所佔的比例，如果你要分割成左右共三部分，你就該寫成<FRAMESET COLS="數字一、數字二,*">的形式。
<FRAME SRC="a3.htm" NAME="a3">中的NAME代表你給這個框架(櫃子)的名字，框架名稱必須不一樣喔。而
FRAME SRC="???.htm" 是框架中目標的網頁檔案。
進階標籤說明。
<FRAMESET COLS="100,*" frameborder="0" framespacing="5">
  • COLS="100,*"
    就是畫面左右分割，比例為100和剩下空間。
  • ROWS="100,*"
    就是畫面上下分割，比例為100和剩下空間。
  • frameborder="0"
    "0"代表不要邊框，"1"代表要邊框。
  • framespacing="5"
    框架與框架間的保留空白的距離，以免太擠。

    <FRAME SRC="a1.htm" NAME="a1" frameborder=0 scrolling="no" noresize marginhight=2 marginwidth=2>
  • SRC="a1.htm"
    設定此框架中要顯示的網頁名稱，每個框架一定要對應一個網頁，否則就會產生錯誤，這裡就是要您填入對應網頁的名稱。（如果該網頁在不同目錄，記得路徑要寫清楚）
  • NAME="1"
    設定這個框架的名稱，這樣才能指定框架來作連結，所以一定要設定喔！當然名稱隨你高興取名。
  • frameborder=0
    設定框架的邊框，"0"代表不要邊框，"1"代表要邊框。
  • scrolling="no"
    設定是否要顯示捲軸，YES是要顯示捲軸，NO是無論如何都不要顯示，AUTO是視情況顯示。
  • noresize
    使用者不能改變這個邊框顯示範圍的大小。
  • marginhight=2
    表示框架高度部份邊緣所保留的空間。
  • marginwidth=2
    表示框架寬度部份邊緣所保留的空間。
    • 網頁的TARGET
    如果我們的網頁a1.htm 中有一個連結，我們希望他的連結內容出現在 a3 的框架中，我們應該輸入
    <a href="連結位址" TARGET="a3">連結文字</A>
    這樣一來我們只要按這個連結，畫面就會出現在 a3 的框架中喔。
