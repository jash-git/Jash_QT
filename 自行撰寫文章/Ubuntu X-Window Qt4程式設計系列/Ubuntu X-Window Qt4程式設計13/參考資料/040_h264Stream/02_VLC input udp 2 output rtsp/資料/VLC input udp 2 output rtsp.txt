RTSP live streaming

Stream with RTSP and RTP:

    * on the server, run: 

% vlc -vvv input_stream --sout '#rtp{dst=192.168.0.12,port=1234,sdp=rtsp://server.example.org:8080/test.sdp}' 

    * on the client(s), run: 

% vlc rtsp://server.example.org:8080/test.sdp
